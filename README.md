NeoVim config
---

Installation:
```bash
mkdir -p ~/.config/nvim
git clone --recursive -j8 git://github.com/perlw/dotvim.git ~/.config/nvim
```
or
```bash
mkdir -p ~/.config/nvim
git clone git://github.com/perlw/dotvim.git ~/.config/nvim
cd ~/.config/nvim
git submodule init
git submodule update
```
